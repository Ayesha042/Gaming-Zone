## 1. Project Overview
Gaming Zone is a web-based gaming website developed using HTML, CSS, and JavaScript. It allows users to log in, play multiple games, track scores, and view a leaderboard. The project demonstrates modular web development and frontend programming skills.

---

## 2. Features
- **Login System:** Users can enter their username to access the dashboard.  
- **Games Included:**  
  - Snake Game (demo)  
  - Quiz Game  
  - Memory Game  
- **Leaderboard:** Tracks and displays user scores.  
- **Score Tracking:** Uses browser LocalStorage to save scores.  
- **Responsive Design:** Works on mobile, tablet, and desktop.  
- **Background Music Support:** (Optional, can add music.mp3)  

---

## 3. Technologies Used
- HTML (for structure)  
- CSS (for styling and responsiveness)  
- JavaScript (for game logic, login, and score management)  

---

## 4. Project Modules
1. **Login Module:** Takes username and stores it locally.  
2. **Dashboard Module:** Displays available games and current user.  
3. **Game Modules:** Each game has its own HTML page and JS logic.  
4. **Leaderboard Module:** Shows total scores from LocalStorage.  

---

## 5. How to Run
1. Download or copy all project files into a folder named `Gaming_Zone_Mini_Project`.  
2. Open `index.html` in any modern browser.  
3. Enter a username and click **Login**.  
4. Navigate to the dashboard to access games.  
5. Scores are automatically tracked and displayed in the leaderboard.
